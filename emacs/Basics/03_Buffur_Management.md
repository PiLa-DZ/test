C-x C-b          --> List Buffers
C-x b Enter      --> Switch Between 2 Buffers 
C-x b TAB        --> Search Buffers
C-x b bufferName --> Open Buffers
    `*Scratch* Buffer`
    `*Message* Buffer`
    `*Warning* Buffer`

C-x left  --> Next Buffer
C-x right --> Previous Buffer

C-x k --> Kill Buffer
